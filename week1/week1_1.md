# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

### 데이터 분석과 데이터 과학
데이터 과학: 통계학, 데이터 분석, 머신러닝, 데이터 마이닝을 아우르는 큰 개념
데이터 분석: 데이터를 수집/처리/정제/분석/모델링하여 의사 결정에 도움을 주는 작업

데이터 분석은 통찰 제공, 데이터 과학은 솔루션 제공

### 데이터 분석을 위한 도구
파이썬: 오픈 소스, 데이터 분석이나 머신러닝에서 표준언어처럼 사용됨
R: 많은 통계 패키지와 그래프를 만들 수 있는 도구를 제공
SQL:데이터베이스 형태일 때 사용함. 다만, 범용적인 프로그래밍 언어는 아님

#### 파이썬 필수 패키지
넘파이: 다차원 배열을 위한 패키지, 과학 계산 중점
판다스: 넘파이와는 달리 숫자와 문자를 섞어 표 형태로 저장하는 데이터프레임을 사용
맷플롯립: 파이썬 데이터 시각화를 위한 기본 패키지
사이파이: 넘파이 기반으로 구축된 수학/과학 계산 전문 패키지
사이킷런: 넘파이와 사이파이에 크게 의존. 가장 널리 사용되는 패키지.

### 데이터 마이닝과 머신러닝
데이터 마이닝: 데이터에서 패턴 혹은 지식을 추출
머신러닝: 데이터에서 자동으로 규칙을 학습하여 문제를 해결하는 소프트웨어를 생성

머신러닝은 데이터 마이닝과 달리 규칙이나 패턴 사용 주체가 사람이 아닌 컴퓨터


## 02. 구글 코랩과 주피터 노트북

### 구글 코랩
코랩은 웹 브라우저에서 파이썬 코드를 작성하고 실행할 수 있게 함

텍스트셀: 셀은 코랩에서 실행하는 최소 단위. HTML과 마크다운을 혼용해서 사용 가능 
코드셀: 파이썬 코드를 입력하고 실행하는 셀. 

일반 프로그램: 처음부터 끝까지 순서대로 모든 코드 실행됨
<-> 노트북: 셀 단위 코드 실행되므로 얼마든지 이전 셀 다시 실행 가능. 순서에 주의할 것!

### 노트북
코랩 노트북은 구글 클라우드의 가상 서버를 사용
동시에 사용할 수 있는 구글 클랑루드의 가상 서버는 최대 다섯개
노트북은 컴퓨터에 다운로드([파일]-[다운로드]) or 구글 드라이브([파일]-[저장])에 저장 가능

### 깃허브
분산 버전 관리 툴인 깃의 저장소를 호스팅하는 웹 서비스. 공동 작업 용이

### 마크다운
HTML과 같이 일반 텍스트로 서식있는 문서 작성


## 03. 이 도서가 얼마나 인기가 좋을까요?

### 코랩에서 데이터 확인하기
CSV 파일 사용: CSV는 콤마(,)로 구분된 텍스트 파일. 한 줄이 하나의 레코드
레코드는 콤마로 구분된 여러 필드로 구성됨

### 파이썬으로 CSV파일 출력하기
open()함수로 읽음. with문으로 파일을 열고 readline() 매서드로 파일에서 한 줄 읽어 출력

#### CSV 파일 인코딩 형식 확인
파이썬 open() 함수는 텍스트 파일이 UTF-8 형식이라고 가정. 그러나 한글 텍스트는 EUC-KR 사용하는 경우가 많음. 따라서 오류 발생 시 형식 확인 필요

chardet.detect() 함수 : 문자 인코딩 방식 확인 가능
open() 함수로 텍스트 파일을 열 때 mode 매개변수를 바이너리 읽기 모드인 'rb'로 지정
-> 문자 인코딩 형식 상관없이 파일 열기 가능

#### 인코딩 형식 지정하기
open()으로 파일을 읽을 때 encoding 메개변수로 인코딩 형식을'EUC-KR'로 지정

### 데이터프레임 다루기: 판다스
판다스는 csv파일을 읽어 데이터프레임 이라는 표 형식 데이터로 저장(행,열로 구성)
시리즈가 1차원 배열이라면, 데이터프레임은 2차원 배열
데이터프레임의 같은 열 속 데이터는 모두 같은 종류여야 함

#### csv파일을 데이터 프레임으로 읽기: read_csv() 함수
자동적으로 파악한 데이터 타입이 달라지면 오류 발생
-> 해결: low_memory 매개변수를 False로 지정해서 파일을 한번에 읽도록 함

head() 매서드로 처음 다섯 개 행 확인
첫번째 열은 데이터프레임의 인덱스
csv파일의 첫 행이 열 이름이 아니라면 header 매개변수를 None으로 지정하고,
names 매개변수에 열 이름 리스트를 따로 전달

##### 판다스의 read_table() 함수
콤마 대신 탭(\t)으로 구분된 파일 읽는 데 사용

### 데이터프레임을 CSV파일로 저장하기: to_csv() 메서드
to_csv() 메서드는 기본적으로 UTF-8 형식으로 저장
저장한 파일을 readline() 메서드를 세번 적는 대신 for 문으로 반복해줌


csv파일에 인덱수가 이미 있다는 것을 알려주려면 index_col 매개변수 사용
혹은 애초에 데이터프레임을 csv파일로 저장할 때 인덱스 빼고 저장 가능
: to_csv() 메서드에 index매개변수를 False로 지정
