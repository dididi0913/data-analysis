# 1️⃣ 개념 정리 

## 01. 통계로 요약하기

describe() 메서드는 수치형 열에 대한 요약 통계를 보여줌
수치가 아닌 다른 데이터 타입의 열의 기술통계를 보고 싶다면  include 매개변수에 데이터 타입 지정.
-> object로 지정한다면, count행은 누락된 값을 제외한 데이터 개수, unique행은 고유한 값의 개수

중복값 제거하고 중앙값 구하기: drop.duplicates()
최솟값, 최댓값 구하기: min(), max()

분위수 구하기: quantile(0.25) - 하위 25% 구하기

분산 구하기: var()

표준편차 구하기: std()

최빈값 구하기: mode() - 텍스트 뿐만 아니라 수치형에도 적용

### 데이터 프레임에서 기술 통계 구하기 
수치형 열만 연산하게끔: numeric_only 매개변수를 True로 지정

### 넘파이의 기술 통계 함수
평균 구하기: mean(), average()
중앙값 구하기: median()
최솟값, 최댓값 구하기: min(), max()
분위수 구하기: quantile()
분산 구하기: var()


## 02. 분포 요약하기

산점도 그리기

맷플롯립은 x축과 y축 값으로 데이터프레임의 열을 입력받을 수 있음

구간 안에 속한 데이터 개수를 도수라고 부름
