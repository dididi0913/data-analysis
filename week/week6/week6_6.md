# 1️⃣ 개념 정리 

## 01. 객체지향 API로 그래프 꾸미기

### pyplot 방식으로 그래프 그리기
plot() 함수에 리스트 형태로 데이터를 전달하고, title() 함수에는 그래프 제목을 넣음

### 출판사별 발행 도서 개수 산점도 그리기
value_counts() 메서드 사용하여 출판사 목록 만듦(슬라이스 연산자 사용)
inin()메서드 사용시, 어떤 행의 출판사가 목록에 있는 출판사 중 하나에 해당하는지 검사 가능
sample()메서드로 데이터프레임의 행 무작위 선택

### 산점도 그리기
x축, y축에 '열' 지정
s 매개변수 사용해서 값에 따라 마커 크기 다르게 나타낼 수 있음
jet(가장 자주 사용됨) 컬러맵을 사용해서 값에 따라 색상 표현 가능
낮은 값일 수록 짙은 파란색이고, 높은 값으로 갈수록 점차 노란색에서 빨간색으로 바뀜


## 02. 맷플롯립의 고급 기능 배우기

### 하나의 피겨에 여러 개 선 그래프 그리기
plot 함수를 여러번 호출
reset_index() : 인덱스 초기화

#### 스택 영역 그래프
하나의 선 그래프 위에 다른 선 그래프를 차례대로 쌓는 것
스택 영역 그래프는 맷플롯립의 stackplot() 메서드로 그릴 수 있음
첫 번째 매개변수에 x축의 값인 '발행년도'를 전달하고, 두 번째 매개변수에는 y축 값을 2차원 배열로 전달
1.pivot_table() 메서드로 각 '발행년도'열의 값을 열로 바꾸기
2.'발행년도'열을 리스트로 바꾸기
3.stackplot()메서드로 스택 영역 그래프 그리기

### 하나의 피겨에 여러 개의 막대 그래프 그리기
선 그래프와 비슷하게 bar() 메서드를 여러번 호출

#### 스택 막대 그래프
위로 쌓아서 여러 막대를 표현
bottom 매개변수 사용하면 수동으로 막대를 쌓을 수 있음

### 데이터값 누적하여 그리기
cumsum() 메서드 사용

### 원그래프 그리기
전체 데이터에 대한 비율을 원의 부채꼴로 나타낸 그래프-> 파이 차트
pie() 메서드 사용
원그래프의 단점: 시각적으로 어떤 데이터가 더 큰지 한눈에 분간 어려움
