# 1️⃣ 개념 정리 

## 01. 맷플롯립 기본 요소 알아보기

### Figure 객체
피겨함수로 피겨 객체를 만들어 활용하면 다양한 그래프 옵션을 조절할 수 있음

그래프 크기 바꾸기: figsize 매개변수
: 튜플로 지정. 기본 그래프 크기는 (6,4)이고 각각 너비와 높이에 해당

그래프 실제 크기 확인하기
: DPI에 따라 화면에 그려지는 그래프 크기가 달라짐
인치 값은 먼저 원하는 픽셀 값을 정한 다음 픽셀 값을 DPI로 나누면 구할 수 있음

그래프 크기 바꾸기: dpi 매개변수

### rcParams 객체
맷플롯립 그래프의 기본값을 관리하는 객체. 객체에 담긴 값을 출력하는 것뿐만 아니라 새로운 값으로 바꿀 수도 있음.

DPI 기본값 바꾸기, 산점도 마커 모양 바꾸기.
기본값 수정하는 것 대신 scatter 함수의 marker 매개변수로 마커 모양 지정하면 더 쉽게 가능

### 여러개의 서브플롯 출력하기
하나의 서브플롯은 두 개 이상의 축을 포함. 

서브플롯 그리기: subplots() 함수
여기서도 피겨 크기를 지정할 수 있는 figsize 매개 변수를 제공. 또한 set_title() 메서드를 사용해서 제목도 넣음

서브플롯을 가로로 나란히 출력하기
subplots() 함수에 행과 열을 지정하면 원하는 서브플롯 개수의 피겨를 만들 수 있음


## 02. 선 그래프와 막대 그래프 그리기

연도별 도서 개수는 value_counts() 메서드로 구할 수 있음
시리즈 객체의 sort_index() 메서드를 사용하면 인덱스 순서대로 데이터를 정렬할 수 있음

주제별 도서 개수를 구할 땐 NaN이라면 -1을 반환하는 함수를 만들어 걸러내야 함

### 선그래프 그리기
그래프 제목은 titile() 함수, x축과 y축 이름은 각각 xlabel()함수와 ylabel()함수를 사용하여 지정

### 선모양과 색상 바꾸기
linestyle 매개변수 사용


### 선 그래프 눈금 개수 조절 및 마커에 텍스트 표시하기 
xticks() 함수 사용
그래프에 값을 표시할 대는 annotate()함수를 사용
텍스트를 마커에서 조금 떼어놓으려면 텍스트 위치를 조절하는 xytest 매개변수를 사용

### 막대 그래프 그리기
텍스트 위치 조절: ha 매개변수에 'center'로 지정
텍스트 크기 조절: fontsize 매개변수
막대의 두께를 조절: width 매개변수


### 가로 막대 그래프 그리기
barh() 함수 사용. 이때 막대의 두께는 hight 매개변수. x축, y축 이름 바꾸어 쓰자

# 3️⃣ 확인 문제

## 문제 1.

> **🧚Q. 다음 데이터를 이용하여 matplotlib으로 선그래프를 그리는 코드를 작성해주세요.**
- x = [1, 2, 3, 4, 5]
- y = [2, 4, 6, 8, 10]
> 조건은 아래와 같습니다.
```
1️⃣ 제목은 "Linear Trend"로 설정해주세요.
2️⃣ x축 이름은 "X values"로 설정해주세요.
3️⃣ y축 이름은 "Y values"로 설정해주세요.
4️⃣ 마커(marker)를 포함하여 선그래프를 그려주세요.
```

```
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

plt.plot(x, y, marker='o')


plt.title("Linear Trend")    
plt.xlabel("X values")     
plt.ylabel("Y values")        
```
