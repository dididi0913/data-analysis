# 1️⃣ 개념 정리 

## 01. 불필요한 데이터 삭제하기

### 데이터 정제
데이터에서 손상되거나 부정확한 부분을 수정하고, 불필요한 데이터를 삭제하거나 불완전한 값을 교체

#### loc 메서드와 불리언 배열
loc 메서드에 슬라이싱 사용하면 원하는 열 첨부터 끝까지 선택 가능

불리언 배열을 사용하면 중간에 있는 열도 제외하고 선택 가능

columns 속성은 판다스의 index 클래스 객체. 이 객체의 원소는 파이썬의 리스트처럼 숫자 인덱스로 참조 가능
판다스 배열 성격의 객체는 어떤 값과 비교할 때 자동으로 배열에 있는 모든 원소와 하나씩 비교해줌

!=(비교연산자) 를 사용하여 반환된 결과는 넘파이 배열

#### drop 메서드
데이터프레임의 행이나 열을 삭제하는 메서드
첫 번째 매개변수에 삭제하려는 열 이름을 전달하고 axis 매개변수를 1로 지정(0은 행을 삭제)
중건에 있는 열도 간단하게 제외 가능

inplace 매개변수를 True로 지정하면 선택한 데이터프레임 바로 수정 가능

#### dropna() 메서드
기본적으로 NaN이 하나 이상 포함된 행이나 열을 삭제
만일, 모든 값이 NaN인 열을 삭제하려면 how 매개변수를 'all'로 지정


### 행 삭제
drop 메서드 사용, axis 매개변소를 0으로 지정

#### 연산자와 슬라이싱
[] 연산자에 슬라이싱이나 불리언 배열을 전달하면 행을 선택함 -> loc메서드에 슬라이싱과는 다르게 마지막 인덱스 포함 X


#### []연산자와 불리언 배열
비교연산자를 활용해 원하는 행은 True로 표시하고 제외할 행은 False로 표시한 불리언 배열을 만들어 사용

### 중복된 행 찾기
duplicated() 메서드 사용
일부 열을 기준으로 중복된 행을 찾으려면 duplicated()메서드의 subset 매개변수에 기준열을 나열

### 그룹별로 모으기 
groupby() 메서드 사용. by 매개변수에는 행을 합칠 때 기준이 되는 열을 지정
만일, dropna=False로 지정하면 연산할 때 NaN이 있는 행도 포함하게 됨

### 원본 데이터 업데이트하기
1. duplicated() 메서드로 중복된 행을 True로 표시한 불리언 배열을 만든다
2. 1에서 구한 불리언 배열을 반전시켜서 중복되지 않은 고유한 행을 True로 표시한다
3. 2에서 구한 불리언 배열을 사용해 원본 배열에서 고유한 행만 선택

중복된 행을 True로 표시한 불리언 배열을 반전시킬 때에는 판다스의 ~연산자를 사용. 
그다음 원본 배열에서 고유한 배열을 선택하여 copy() 메서드로 데이터프레임을 만듦


#### 원본 데이터프레임 인덱스 설정하기
지정한 열을 인덱스로 설정할 때는 set_index() 메서드를 사용
이때 inplace 매개변수를 True로 지정해 새로운 데이터프레임을 반환하지 않고 ns_book3 데이터프레임을 수정. 

#### 업데이트하기:update() 메서드
reset_index() 메서드로 데이터프레임 인덱스 재설정 가능

### 일괄 처리 함수 만들기
열을 삭제: loc 메서드에 슬라이싱이나 불리언 배열을 적용 / drop()메서드를 사용
행을 삭제: [] 연산자와 슬라이싱 또는 불리언 배열 사용
중복된 행 찾기: groupby() , sum(), set_index(), reset_index() 메서드 사용

#### 일괄 처리 함수
필요한 코드를 파이썬 함수로 만들어 저장하여 간단하게 함수를 호출
다른 데이터프레임과 비교할 때는 equals() 메서드 사용



## 02. 잘못된 데이터 수정하기
NaN을 확인하고, 그걸 채우는 여러가지 방법

### 데이터프레임 정보 요약 확인하기
info() 메서드를 사용하면 열마다 NaN이 아닌 값이 얼마나 있는지 확인 가능

### 누락된 값 처리하기

#### 누락된 값 개수 확인하기: isna() 메서드

누락된 값으로 표시하기: None과 np.nan
데이터 값을 지정할 때, astype() 메서드를 사용

##### 누락된 값 바꾸기(1) : loc, fillna() 메서드
loc 메서드를 사용하면 누락된 값을 원하는 값으로 바꿀 수 있음
혹은, fillna() 메서드에 원하는 값을 전달하면 NaN을 대체할 수 있음

#### 누락된 값 바꾸기(2) : replace() 메서드
replace() 메서드는 NaN은 물론 어떤 값도 바꿀 수 있는 편리한 메서드

바꾸려는 값이 하나일때, 첫번째 매개변수에는 원래 값을, 두 번재 매개변수에는 새로운 값을 전달

바꾸려는 값이 여러개일 때, 리스트 형식으로 전달

열마다 다른 값으로 바꿀 때, 열 이름과 바꾸려는 값을 딕셔너리 형식으로 전달

### 정규 표현식
정규식은 문자열 패턴을 찾아서 대체하기 위한 규칙의 모음

- 숫자 찾기: \d

regex 매개변수 옵션: 정규식 사용할 때. replace() 메서드와 아주 비슷. 
정규 표현식 앞에 붙인 r문자는 파이썬에서 정규식을 다른 문자열과 구분하기 위해 접두사처럼 붙임

- 문자 찾기: 마침표(.)

반복 개수를 지정하기 어려울 때는, *문자를 사용하여 0개 이상 반복된다고 표시.
역슬래시(\)를 붙이면 괄호를 일반 문자라고 인식하게 함
정규표현식 \s는 공백 문자를 나타냄


### 잘못된 값 바꾸기

contains()메서드: 시리즈나 인덱스에서 문자열 패턴을 포함하고 있는지 검사. 기본적으로 정규 표현식을 인식
이 메서드에서 na매개변수를 True로 지정하여 연도가 누락된 행을 True로 표시.

gt() 메서드: 전달된 값보다 큰 값을 찾음


### 누락된 정보 채우기
뷰티플수프를 사용함

도서의 저자는 두명 이상일 수 있기 때문에 뷰티플수프의 find_all() 메서드를 사용-> 저자를 담은 <a> 태그 모두 추출
추출 결과를 join() 메서드를 사용해 하나의 문자열로 합쳐줌
re 모듈의 findall() 함수를 사용하면 원하는 정규식에 매칭되는 모든 문자열을 찾아 리스트로 반환해줌

오류 떄문에 함수 실행 종료안하고 계속 다음 요소 추출하도록 try~except 문으로 예외 처리
