# 1️⃣ 개념 정리 

## 01. 통계적으로 추론하기

### 모수검정이란
모집단에 대한 파라미터를 추정하는 방법
모집단에서 선택한 일부 샘플은 표본이라고 부름
모수검정은 모집단 데이터에 대한 어떤 검정을 전제로 하고 수행되곤 함

### 누적분포 이해하기
평균이 0이고, 표준편차가 1인 정규분포를 표준정규분포라고 함

### 중심극한정리
'무작위로 샘플을 뽑아 만든 표본의 평균은 정규분포에 가깝다'는 이론
샘플링 크기를 늘릴 수록 평균에 가까워짐
표본 평균의 표준오차가 모집단의 표준편차를 표본 크기의 제곱근으로 나눈 것에 가깝다

### 모집단의 평균 범위 추정하기: 신뢰구간
신뢰구간은 표본의 파라미터가 속할 것이라고 믿는 모집단의 파라미터 범위

### 통계적 의미 확인하기: 가설검정
가설검정은 표본에 대한 정보를 사용해 모집단의 파라미터에 대한 가정을 검정하는 것
영가설(귀무가설): 표본 사이에 통계적으로 의미가 없다고 예상되는 가설
대립가설: 표본 사이에 통계적인 차이가 있다
유의수준: 정규분포의 양쪽 꼬리 면적을 더해 5%가 되는 지점

### t검정으로 가설 검증하기
t분포는 정규분포와 비슷하지만, 중앙은 조금 더 낮고 꼬리가 더 두꺼운 분포
표본 크기가 30 이하일 때 쓰는 게 좋고, 30 이상이면 정규분포와 매우 비슷해짐

### 정규분포가 아닐 때 가설 검증하기: 순열검정
순열검정은 모집단의 분포가 정규분포가 아니거나, 모집단의 분포를 알 수 없을 때 사용 -> 비모수검정
두 표본의 평균 차이를 계산한 후, 두 표본을 섞고 무작위로 두 그룹으로 나눔. 이 나눈 그룹에서
다시 평균 차이를 계산. 여러번 반복해서 원래 표본의 평균 차이가 무작위로 섞은 그룹의 평균 차이보다 크거나 작은 경우를 헤아려 p값을 계산




## 02. 머신러닝으로 예측하기

머신러닝은 컴퓨터 프로그램을 이용해 데이터에서 패턴을 학습하는 방법
머신러닝은 인공지능의 하위 분야. 지도학습과 비지도학습으로 나뉨

### 모델
머신러닝으로 학습된 패턴을 저장하는 소프트웨어 객체

### 지도학습과 비지도학습
지도학습은 데이터에 있는 각 샘플에 대한 정답을 알고 있음. 정답 = 타깃
비지도학습은 입력 데이터는 있지만 타깃이 없는 경우. 

### 모델 훈련하기
입력A, 타깃A = 훈련 세트
입력B, 타깃B = 테스트 세트
train_test_split() 함수로 훈련 세트/테스트 세트 나눔

훈련된 모델을 평가할 땐, score() 메서드를 사용해서 결정계수를 계산함
타깃에 정확히 맞는 예측을 하면 이 결정계수가 1에 가까워짐

### 연속적인 값 예측하기:선형 회귀
기울기, y절편을 더하여 예측을 만듦

### 카테고리 예측하기: 로지스틱 회귀
타깃이 실수가 아니라 어떤 종류의 카테고리일 때.
두개의 가테고리-> 이진 분류
세개 이상의 카테고리-> 다중 분류
로지스틱 회귀는 선형 함수를 사용하지만, 예측을 만들기 전에 로지스틱 함수를 거침. 
연속적인 실수 출력값을 1또는0으로 변환

## 문제 1.

> **🧚Q. 어떤 표본의 크기가 충분히 크고(n ≥ 30), 모집단 분산을 알 수 없을 때 모평균에 대한 신뢰구간을 구하려고 한다.
이때 다음 설명 중 가장 적절한 것은 무엇인가?**

```
1️⃣ 표본의 크기가 충분히 크므로 중심극한정리에 의해 표본평균의 분포는 근사적으로 정규분포를 따른다.  
2️⃣ 모집단 분산을 모르므로 신뢰구간을 구할 수 없다.  
3️⃣ 표본이 30개 이상이면 항상 모집단도 정규분포를 따른다.  
4️⃣ 중심극한정리는 모집단이 정규분포일 때만 적용된다.
```

```
1번
```
